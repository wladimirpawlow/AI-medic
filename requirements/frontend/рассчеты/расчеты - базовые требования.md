# Общие данные
На экране должна отображаться таблица с полным перечнем всех расчетов, которые были созданы в системе.

Описание таблицы :
| номер столбца | имя столбца | описание | тип данных |
|-|-|-|-|
|1| запуск | идентификатор запуска | UUID |
|2| набор/осмотр | наименование набора/ID осмотра | строка |
|3| версия | версия набора (только для запусков по набору, для запусков по осмотру - пусто) | натуральное число |
|4| алгоритм | версия алгиритма расчета | строка |

|5| дата расчета | дата расчета | время-дата |
|6| идентификатор | имя файла расчета и ссылка для скачивания файла расчета | строка |
