# Общие данные
Экран должен состоять из трех секций:
- общие данные по набору
- перечень запусков по набору
- перечень осмторов в ходящих в набор

#  общие данные по набору

Данная секция должна содержать следующую информацию:
- идентификатор набора
- наименование набора
- текущая версия набора
- количество осмотров в наборе
- дата последнего изменения

Данные параметр-значение должны быть выведены один по другим.


# перечень запусков по набору
Данная секция должна содержать:
- таблицу, полность повторяющую таблицу экрана "запуски" без столбца "ид осмотра" и без фильтров. В таблицу входят только запуски, по данному набору
- кнопку "запустить анализ". При нажатии открывается модальное окно с выпадающим списком версий САВ и кнопками "запустить"/"отменить"

# перечень осмторов в ходящих в набор
Данная секция должна содержать:
- таблицу, полность повторяющую таблицу экрана "осмотры" без столбца "дата загрузки" и без фильтров. В таблицу входят только осмотры, входящие в данный набор
- кнопку "добавить". При нажатии открывается модальное окно с выпадающим списком осмторов и кнопками "добавить"/"отменить"
