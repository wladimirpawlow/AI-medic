# Общие данные
На экране должна отображаться таблица с полным перечнем всех наборов, которые были созданы в системе.

Описание таблицы :
| номер столбца | имя столбца | описание | тип данных |
|-|-|-|-|
|1| id | внутренний идентификатор набора | UUID |
|2| наименование | наименование набора | строка |
|3| версия | текущая версия набора | натуральное число |
|4| количество осмотров | количество осмотров в наборе | натуральное число |
|5| дата изменения | дата последнего изменения | дата-время |

Принажатии строку таблицы должен осуществляется переход на экран данного набора.


# Создание наборов
На экране должен быть реалзован элемент (кнопка) "создать набор".
При нажатии на кнопку должно открываться модальное окно со следуюшими функциями:
- поле ввода "имя набора"
- кнопка "содать"
- кнопка "отменить"

# Удаление наборов
В конце каждой строке таблицы должна быть иконка "удалить". При нажатии открывается модальное окно подтверждения действия
