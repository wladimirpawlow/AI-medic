Приложение предназначено для работы следующих групп пользователей:
1. администратор
2. эксперт
3. пользователь

Функциональные возможности системы:
1. загрузка медосмотров из внешних источников
2. просмотр перечня загруженных медосмотров
3. просмотр деталей загруженных данных по каждому из загруженных медосмотров
4. формирование наборов медосмотров
5. запуск обработки медосмотров в САВ:
- для единичного медосмотра
- для набора
- повторный запуск по одному из предыдущих запусков
6. просмотр всех запусков и их текущего статуса
7. просмотр детальных результатов анализа получинных от САВ для выбранного осмотра
8. просмотр решений для выбранного осмотра по выбранному алгоритму рассчета
9. форммирование расчета решений для выбранного запуска и их сохраненние в виде табличного файла отчета
10. просмотр всех сформированных расчетов
11. выгрузка выбранного рассчета

Структура интерфейса.
Интерфейс должен содержать следующие экраны:
1. осмотры - таблица всех загруженных осмотров с возможностью филльтрации
2. наборы - таблица всех сформированных наборов с возможностью фильтрации
3. запуски - таблица всех запусков с возможностью фильтрации
4. расчеты - таблица всех сформированных расчетов с возможностью фильтрации
5. осмотр - детальная страница осмотра с возможностью просмотра деталей осмотра
6. набор - детальная страница набора с возможностью просмотра всех осмотров в рамках набора и корректировки перечня

Общие требования.
Дизайн должен бть реализован в минималистичном стиле для корпоративных систем.
Предпочтительны светлие серые, голубые и синие цвета.
