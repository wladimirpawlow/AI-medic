На экране должна отображаться таблица с полным перечнем всех осмотров, которые были загружены в систему.
Таблица должна содржать возможность пагинации с отображением 10/20/50/100 запусков.
Таблица должна содержать поиска/фильтры по каждому столбцу.
Для столбцов с датами должна быть реализована возможность фильтров по диапазонам значений.

Таблица экрана осмотров должна содержать следующие данные (столбцы):
1. id - внутренний идентификатор медосмотра
2. внешний id - внешний идентификатор медосмотра
3. контрагент - идентификатор контрагента
4. id сотрудника- идентификатор сотрудника
5. id организации - идентификатор организации 
6. id МР - идентификатор медработника проводившего осмотр во внешней системе
7. id мед-орг - идентификатор мед-организации, проводившей осмотр
8. id ПАК - идентификатор ПАК
8. тип ПАК - тип ПАК (стационарный/мобильный)
9. эталон - наличие эталон
10. наличие замечаний МР - иконки замечаний
11. дата загрузки

Принажатии на каждый осмотр осуществляется переход на экран осмотра
