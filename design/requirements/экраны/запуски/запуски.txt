На экране должна отображаться таблица с полным перечнем всех запусков, которые были сформированы.
Таблица должна содржать возможность пагинации с отображением 10/20/50/100 запусков.
Таблица должна содержать поиска/фильтры по каждому столбцу.
Для столбцов с датами должна быть реализована возможность фильтров по диапазонам значений.

Таблица экрана запусков должна содержать следующие данные (столбцы):
1. ИД запуска
2. версия САВ
3. ИД набора и номер версии набора, если запуск по набоу
4. ИД осмотра, если запуск по осмотру
5. дата/время начала запуска
6. текущий статус запуска (N/M завершенных обработок из набора, 1/1 для запуска по осмотру)
7. дата/время завершения запуска
8. инициатор старта запуска

Для каждого незавершенного запуска должна быть предусмотрена возможность остановить процесс обработки
Для каждого завершенного запуска предусмотреть функционал запуска расчета решений по данному запуску
