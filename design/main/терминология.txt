Термины и определения:

1. замечания - признаки, наличие/отсутствие которых должно быть определено
2. исходные данные - набор фото и видеоматериалов по медосмотру для обработки САВ

3. САВ - сервис аналитики видео
4. эксперт - "доверенный" медицинский работник, чье решение считается эталоном (достоверным)
5. эталон - решение по наличию/отсутствию замечания на записи, которое считается достоверным

6. набор - ряд медосмотров объединенных группу. Используется для формирования статистики. Один осмотр может входить в несколько групп одновременно
7. запуск - выполнение анализа обработки видео в САВ
8. результат анализа - значения вероятностей по данному признаку, полученное из САВ

9. решение - значение ДА/НЕТ/Н.О. для данного замечания в рамках данного осмотра (в случае САВ для данного результата анализа по заданным параметрам обработки)
10. алгоритм/параметры обработки -  алгоритмы, наборы отсечек, уставок и т.п., которые используются для формирования решений
11. источник решения - МР/ДМР/ИИМ-v(версия ИИМ учитывает: версии ядра САВ, параметров обработки)
