# Frontend

React приложение для системы управления 

## Структура приложения

- **Меню** - статическое меню в верхней части экрана с кнопками навигации
- **Страницы**:
  - Замечания (`/remarks`)
  - Осмотры (`/inspections`)
  - Статистика (`/statistics`)
  - Выборки (`/selections`)

## Установка и запуск

### Локальная разработка

```bash
npm install
npm start
```

Приложение будет доступно по адресу `http://localhost:3000`

### Сборка для production

```bash
npm run build
```

### Запуск через Docker

```bash
docker-compose up --build frontend
```

Frontend будет доступен по адресу `http://localhost:3000`

## Особенности

- Статическое меню доступно на всех экранах
- Выпадающее меню логина с информацией о пользователе
- Роутинг между страницами через React Router
- Готовность к развертыванию в Docker
